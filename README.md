# 📊 Power BI Sales Dashboard Project

This repository contains an end-to-end Power BI project where I performed data cleaning, transformation, created insightful DAX measures, and built an interactive dashboard to showcase key sales metrics and business performance indicators.

---

## 🔧 Project Overview

This project simulates a real-world business scenario where stakeholders require insights into:

- Total sales performance  
- Regional and category-level trends  
- Customer engagement metrics  
- Monthly performance comparisons  

The dataset was cleaned, transformed, and modeled using **Power BI Power Query Editor**, followed by the creation of **custom DAX measures** to generate deeper business insights.

---

## 🛠 Tools & Technologies

- **Power BI Desktop**
- **Power Query** (for ETL operations)
- **DAX** (for calculated measures)
- **Data Modeling**
- **Data Visualization** (Slicers, KPIs, Charts, Maps)

---

## 📈 Key Features

### KPI Cards
- Total Sales
- Total Orders
- Customer Count

### Custom DAX Measures
- Monthly Growth Rate
- Sales per Customer
- Category Contribution %
- Year-over-Year (YoY) Comparisons

### Visualizations Used
- Stacked Bar Charts
- Line & Area Charts
- Donut and Pie Charts
- Interactive Slicers

---

## ✨ Dashboard Highlights

- 🚀 Cleaned and transformed raw sales data using **Power Query**
- 🧠 Created calculated columns and measures using **DAX**
- 🌍 Visualized regional sales distribution using **map visuals**
- 📊 Compared sales and order volume across **categories and segments**
- 📅 Performed **monthly trend analysis** using time intelligence functions

---

## 🧪 How to Use

1. Download the file **Sales_Overview.pbix** from this repository  
2. Open the file using **Power BI Desktop**  
3. Explore the dashboard using filters, slicers, and visual interactions
